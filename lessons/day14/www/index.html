Project Summary

This website is stored in a private S3 bucket and delivered through CloudFront edge locations. Terraform provisions the full stack so deployments are consistent, repeatable, and easy to update.

Amazon S3 (Private)

Static files are stored securely. Public access is blocked to protect the origin.

  • Private bucket
  • Versioning-ready
  • Origin-only access
⚡

CloudFront CDN

CloudFront caches content at the edge to improve performance worldwide.

  • Edge caching
  • HTTPS by default
  • Low latency
🔒

OAC + Bucket Policy

Only CloudFront can read objects from S3. Users never hit the bucket directly.

  • Origin Access Control
  • Locked-down S3 origin
  • Reduced attack surface
🏗️

Terraform IaC

Infrastructure is created from code, making builds consistent and auditable.

  • Repeatable deployments
  • Simple updates
  • Tagging + structure

What This Design Achieves

  • Fast: Edge caching reduces load time.
  • Secure: S3 stays private using OAC + bucket policy.
  • Simple: No servers to manage.
  • Repeatable: Terraform rebuilds the same stack anytime.
